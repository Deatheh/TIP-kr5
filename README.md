<h2><b>Таймер обратного отсчёта</b></h2></br>
<p>
  Приложение запускается на 3000 порту <br>
  <b>Index</b><br>
  [get] http://localhost:3000/ - начальная страница сайта, где можно выбрать начальное время таймера<br>
  [post] http://localhost:3000/start - запрос-редирект на страницу с таймером<br>
  <b>Timer</b><br>
  [get] http://localhost:3000/timer - страница с таймером. Принимает query параметр second<br>
</p>
